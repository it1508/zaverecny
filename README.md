# zaverecny
- informacni aplikace v androidu pro hru Warframe
- Ispiraci pro tento projekt jsem dostal při hraní hry ktérá neměla žádnou informační aplikaci na mobil která by dokázala ukázat jakékoliv informace o předmětech ve hře. Účelem této aplikace je urychlit hledání informací než chodit na internet a složitě je vyhledávat už tak v složitém webdesignu.
